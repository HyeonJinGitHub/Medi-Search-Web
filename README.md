### 개요
* 약 조회 및 해당 약 정보를 사용자가 개인 페이지에 등록하여 정보를 받아볼 수 있는 웹 서비스
  * 회원가입, 개인정보 수정, 약 검색, 개인 약 등록 및 삭제 등의 기능
  * MVC 패턴 사용
